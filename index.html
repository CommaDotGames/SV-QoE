Dataset Introduction

This dataset consists of stereoscopic and 2D video sequences generated entirely in Unity Engine (v.6000.0.27f1). The virtual camera setup and rendering parameters are as follows:

  • Stereo baseline: 65 mm (average human interpupillary distance)
  • Camera positions:
    • Left_Camera: X = −0.325 m, Y = 0, Z = 0
    • Right_Camera: X = +0.325 m, Y = 0, Z = 0
    • Central_Camera: X = 0, Y = 0, Z = 0
  • Field of View: 137° (horizontal)
  • Resolution & Frame Rate: 4K (3840 × 2160), 60 FPS
  • Clipping planes: Near = 0.3 units, Far = 1000 units
  • Depth priority: -1
  • Rendering features: HDR + MSAA enabled, occlusion culling on, render path: Use Graphics Settings
  • Compression: MV-HEVC (.MOV), multi-quality x265 CRF (0–51)
